# Thumbnail CTR Optimization Checklist

**Purpose:** Validate thumbnail has all CTR-boosting elements before publication
**Agent:** Paddy Galloway (YouTube Thumbnail Strategist)
**Standard:** Data-driven CTR optimization based on Glance Test Framework

---

## GLANCE TEST (0.5 SECONDS)

> "You have 0.5 seconds. Half a second. That's your entire window."

### Layer 1: Instant Recognition (0-100ms)

- [ ] Tema/assunto is IMMEDIATELY clear
- [ ] No visual confusion or overlapping elements
- [ ] Someone unfamiliar could identify topic in <100ms
- [ ] **Score: ___/3**

### Layer 2: Emotional Response (100-300ms)

- [ ] Clear emotional reaction is triggered
- [ ] Emotion is NOT neutral or bland
- [ ] Human element (face/expression) supports emotion
- [ ] **Score: ___/3**

### Layer 3: Curiosity Gap (300-500ms)

- [ ] Information gap creates "need to click" feeling
- [ ] Thumbnail does NOT reveal everything
- [ ] Question or mystery is implied
- [ ] Viewer cannot satisfy curiosity without clicking
- [ ] **Score: ___/4**

**TOTAL GLANCE SCORE: ___/10**

| Score | Action |
|-------|--------|
| 8-10 | Ready to publish |
| 6-7 | Minor adjustments needed |
| 4-5 | Refazer conceito |
| 0-3 | Start from scratch |

---

## EMOTION HIERARCHY

> "Curiosity creates clicks. Satisfaction kills them."

- [ ] Primary emotion is clearly identifiable
- [ ] Emotion matches content type (see table below)
- [ ] Emotion is NOT "felicidade generica" (weakest CTR)
- [ ] Emotion is visible at mobile thumbnail size (160x90)

**Emotion Selection Guide:**

| Content Type | Best Emotion | CTR Boost |
|--------------|--------------|-----------|
| Tutorial | Curiosidade + Aspiracao | +100-150% |
| Review | Controversia + Curiosidade | +100-200% |
| Story/Vlog | Curiosidade | +150-300% |
| Opinion | Controversia | +100-200% |
| Educational | Curiosidade + leve Medo | +100-200% |

## FACE & EXPRESSION

> "Faces are click magnets. The human brain is wired to notice faces before anything else."

### If Face is Present:

- [ ] Face occupies 30-50% of thumbnail (not too small, not too big)
- [ ] Expression is EXTREME, not subtle
- [ ] Expression matches selected emotion
- [ ] Expression is NOT sorriso generico
- [ ] Face is positioned off-center (rule of thirds)

**Expression Check:**

| Expression | CTR Impact | This Thumbnail? |
|------------|------------|-----------------|
| Choque/Surpresa | +100-200% | [ ] |
| Curiosidade | +50-100% | [ ] |
| Raiva/Frustracao | +50-100% | [ ] |
| Determinacao | +30-50% | [ ] |
| Sorriso Generico | +0-20% | [ ] AVOID |
| Neutra | -20% | [ ] NEVER |

### Eye Direction:

- [ ] Eyes are directing viewer attention appropriately
- [ ] Direct camera: Personal connection, opinions
- [ ] Looking at object: Reviews, tutorials
- [ ] Off-camera: Mystery, storytelling

## TEXT OPTIMIZATION

> "If you can say it with an image, don't use text. If you need text, use 3 words or less."

### The 3-Word Rule:

- [ ] Text is 3 words or fewer
- [ ] OR thumbnail has NO text (image-only)
- [ ] Text does NOT repeat the video title
- [ ] Each word adds essential value

### Text Type Check:

| Type | Impact | This Thumbnail? |
|------|--------|-----------------|
| Numbers ($1M, 30 DIAS) | HIGHEST | [ ] |
| Questions (COMO?) | HIGH | [ ] |
| Negatives (NAO, ERRO) | HIGH | [ ] |
| Superlatives (MELHOR) | MEDIUM | [ ] |
| Descriptive (Tutorial) | LOW | [ ] AVOID |
| No Text | VARIES | [ ] |

**Text Used:** _______________

### Typography:

- [ ] Font is bold sans-serif (Impact, Bebas Neue, Anton)
- [ ] Text size is 15-25% of thumbnail height
- [ ] Text has stroke/shadow for contrast
- [ ] Text is readable at 160x90px (mobile)

---

## COLOR & CONTRAST

> "Contrast is king. Colors that clash get clicks."

### Contrast Check:

- [ ] Primary colors have HIGH contrast
- [ ] Text is clearly readable against background
- [ ] Thumbnail does NOT blend into YouTube feed
- [ ] Tested in both dark mode AND light mode

### Color Combination:

| Combination | Contrast Level | CTR Impact |
|-------------|---------------|------------|
| Yellow/Black | Maximum | +50-100% |
| Red/White | Very High | +50-100% |
| Blue/Orange | High | +30-50% |
| Green Neon/Black | High | +30-50% |

**Colors Used:** _______________

### Visual Check:

- [ ] Thumbnail "pops" when viewed in context of feed
- [ ] Skin tones contrast with background (if face present)
- [ ] No YouTube red (competes with UI)

---

## CURIOSITY ELEMENTS

> "The best thumbnail creates a question that only the video can answer."

- [ ] There is an incomplete story that demands clicking
- [ ] Information gap is clear (not vague)
- [ ] Viewer feels they MUST know more
- [ ] Thumbnail does NOT "give away" the answer

**Curiosity Mechanism Used:**

- [ ] Partial reveal (something hidden/blurred)
- [ ] Before/after with "after" implied
- [ ] Question (visual or text)
- [ ] Number without context (creates "why?")
- [ ] Contrast/comparison implied
- [ ] Reaction to unseen element

---

## TITLE-THUMBNAIL ALIGNMENT

> "Thumbnail and title are ONE system. They complement, they don't repeat."

- [ ] Thumbnail does NOT repeat the title
- [ ] Thumbnail COMPLEMENTS the title
- [ ] Thumbnail emotion matches title promise
- [ ] Together they tell one cohesive story
- [ ] Information is distributed (not duplicated)

---

## ABC TESTING READINESS

> "The thumbnail you think will win almost never does. Test everything."

- [ ] This thumbnail is NOT the only variant
- [ ] At least 4 variants are ready (ideal: 10-20)
- [ ] Variants are DRASTICALLY different (not subtle tweaks)
- [ ] Backup variants are pre-uploaded or ready
- [ ] Swap trigger CTR threshold defined: ____%
- [ ] Monitoring schedule confirmed

---

## MOBILE-FIRST CHECK

> "If it doesn't work at 160x90 pixels, it doesn't work."

- [ ] Thumbnail viewed at actual mobile size
- [ ] Face/expression still visible at small size
- [ ] Text still readable at small size
- [ ] Main elements not lost in small format
- [ ] Curiosity element still clear at small size
