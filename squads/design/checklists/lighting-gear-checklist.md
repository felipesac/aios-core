# Lighting Gear Checklist

**Purpose:** Pre-shoot gear checklist for lighting setups
**Agent:** Joe McNally (Photography Lighting Expert)
**Standard:** Professional on-location and studio lighting

---

## PRE-SHOOT GEAR VERIFICATION

Complete before leaving for any shoot. Check each item and note quantity/status.

---

## LIGHT SOURCES

### Speedlights
- [ ] Primary speedlight (charged, tested)
- [ ] Backup speedlight (charged)
- [ ] Third speedlight (if multi-light setup)
- [ ] All flashes on same firmware version
- [ ] Flash heads clean (no dust on sensor)

### Strobes (if applicable)
- [ ] Strobe heads (tested)
- [ ] Power packs (charged if battery)
- [ ] Cables (tested, no damage)
- [ ] Modeling lamps working

### Continuous (if applicable)
- [ ] LED panels (charged)
- [ ] Brightness adjustment working
- [ ] Color temperature adjustment working
